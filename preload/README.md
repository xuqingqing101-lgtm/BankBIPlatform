# 预置数据目录

此目录用于存放自动加载到系统的初始数据。

## 目录结构

- `knowledge/`: 存放知识库文档（支持 .txt, .pdf, .docx, .ppt, .xls 等）。
  - 文档会自动上传并赋予 ADMIN 和 USER 角色权限。
- `data/`: 存放结构化数据（支持 .csv）。
  - CSV 文件名将作为表名（例如 `sales_2023.csv` -> 表名 `sales_2023`）。
  - 系统会自动推断列类型并建表。

## 使用说明

1. 将文件放入对应目录。
2. 重启后端服务，或者调用 API `/api/system/preload` 手动触发加载。
