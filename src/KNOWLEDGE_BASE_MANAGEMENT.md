# 知识库完整管理功能说明

## 🎯 功能概述

知识库现在拥有完整的文档管理功能，包括**上传、编辑、打标签、删除、搜索、分类、问答**等全方位的知识管理能力。

## ✨ 核心功能

### 1. 📤 文档上传

**入口：** 文档管理 → 上传文档按钮

**功能：**
- 📝 填写文档名称（必填）
- 📁 选择文件上传（支持 PDF、Word、Excel、PPT）
- 🏷️ 选择分类（必填）
- 🏷️ 添加标签（多个标签逗号分隔）
- 📄 添加描述信息

**支持格式：**
- PDF (.pdf)
- Word (.doc, .docx)
- Excel (.xls, .xlsx)
- PowerPoint (.ppt, .pptx)
- 最大文件大小：50MB

**上传流程：**
```
点击"上传文档" → 填写信息 → 选择文件 → 点击"上传" → 完成
```

**字段说明：**
- **文档名称**：显示在列表中的名称
- **分类**：根据当前知识库自动提供选项
- **标签**：用于搜索和筛选，支持多个
- **描述**：简要说明文档内容（可选）

### 2. ✏️ 文档编辑

**入口：** 文档管理 → 文档列表 → 编辑按钮（铅笔图标）

**可编辑项：**
- ✅ 文档名称
- ✅ 分类
- ✅ 标签

**编辑流程：**
```
点击编辑按钮 → 修改信息 → 点击"保存" → 完成
```

**注意事项：**
- 不能修改文件本身（需重新上传）
- 修改后立即生效
- 支持批量修改标签

### 3. 🏷️ 打标签

**入口：** 文档管理 → 文档列表 → 打标签按钮（标签图标）

**功能特性：**
- ✅ 手动输入标签（逗号分隔）
- ✅ 快速添加常用标签（一键添加）
- ✅ 删除已有标签（点击 X）
- ✅ 实时预览当前标签

**常用标签：**
- 对公
- 零售
- 贷款
- 存款
- 反洗钱
- 合规
- 风控
- 营销
- 产品
- 培训

**打标流程：**
```
点击打标签 → 添加/删除标签 → 点击"保存" → 完成
```

**标签用途：**
- 🔍 搜索和筛选
- 📊 分类统计
- 💡 智能推荐
- 🔗 关联文档

### 4. 🗑️ 文档删除

**入口：** 文档管理 → 文档列表 → 删除按钮（垃圾桶图标）

**删除流程：**
```
点击删除按钮 → 确认删除 → 完成
```

**安全机制：**
- ⚠️ 删除前弹出确认对话框
- ⚠️ 删除后无法恢复
- 📋 建议先下载备份

### 5. 🔍 智能搜索

**入口：** 智能检索标签页

**搜索功能：**
- 📝 关键词搜索
- 🏷️ 按分类筛选
- 🏷️ 按标签筛选

**搜索范围：**

- 分类

**快速筛选：**

**全员知识库：**

**机构知识库：**

## 🚀 使用流程

### 场景1：上传新文档
```
1. 选择目标知识库（全员/合规/业务/机构）
2. 进入"文档管理"标签页
3. 点击"上传文档"按钮
4. 填写文档信息：
   - 文档名称：对公贷款审批流程
   - 分类：业务指南
   - 标签：对公,贷款,审批

7. 点击"保存"

## ⚠️ 注意事项

### 上传限制
- 单个文件最大 50MB
- 仅支持文档类型（PDF、Office）
- 文件名建议使用中文，便于搜索

### 删除警告
- ⚠️ 删除操作不可逆
- ⚠️ 建议先下载备份
- ⚠️ 确认无人使用后再删除

### 标签规范
- ✅ 使用常用词汇
- ✅ 避免重复标签
- ✅ 不超过10个标签
- ✅ 用逗号或空格分隔

### 分类选择
- 📁 根据文档内容选择合适分类
- 📁 一个文档只能属于一个分类
- 📁 分类决定文档的展示位置

```
