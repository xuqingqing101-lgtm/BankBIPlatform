# 企业智能AI助手平台 Demo

## 📋 概述

这是一个基于 **Deepseek 大模型**的企业智能AI助手平台演示，专为企业经营、财务、贸易和物流管理设计。平台采用对话式AI助手界面，用户可以通过自然语言查询各类业务数据，获得智能分析和决策建议。

## 🎯 核心功能

### 1️⃣ 智能对话问答
- **自然语言查询**：无需学习复杂的报表系统，直接用日常语言提问
- **多数据源整合**：自动从NC财务系统、OA系统、大宗贸易平台等多个数据源获取信息
- **智能数据分析**：不仅给出数据，还提供深度洞察和优化建议

### 2️⃣ 实时数据看板
- **经营驾驶舱**：营收、利润、项目数量、客户数等核心KPI
- **财务指标**：资产负债率、流动比率、净利润率等财务健康度
- **贸易分析**：客户信用、交易情况、合同风险等
- **物流运营**：运输成本、应收账款、在途车辆等

### 3️⃣ 可视化展示
- **动态图表**：自动生成趋势图、柱状图、饼图等可视化内容
- **数据表格**：清晰展示明细数据和对比分析
- **KPI卡片**：关键指标一目了然

### 4️⃣ 智能洞察
- **AI深度分析**：每次查询都附带智能洞察和建议
- **风险预警**：自动识别经营风险和异常情况
- **趋势预测**：基于历史数据给出趋势判断

## 🔧 技术特点

### AI助手体验
- **对话式界面**：类似ChatGPT的聊天体验，降低使用门槛
- **上下文理解**：记住对话历史，支持连续提问
- **智能推荐**：根据用户角色推荐相关问题

### 现代化UI设计
- **深色主题**：专业科技感的视觉设计
- **流畅动画**：平滑的过渡和交互效果
- **响应式布局**：支持桌面和移动设备

### 数据集成
- ✅ NC财务系统（财务报表、成本核算）
- ✅ 大宗贸易平台（客商、项目管理）
- ✅ OA办公系统（知识库、档案）
- ✅ 物流管理系统（运输、仓储）
- ✅ 合同管理系统（合同审核）

## 💡 使用场景示例

### 场景1：高管查询经营数据
**提问**：本季度利润比上季度增长了多少？

**AI回答**：
- 展示季度利润趋势图表
- 分析利润增长的主要来源（如粮食贸易板块）
- 指出需要关注的项目（如西南大豆运输利润率低）
- 给出优化建议（如优化成本结构）

### 场景2：业务员查询客户信息
**提问**：帮我查看华东粮油的信用额度使用情况

**AI回答**：
- 显示客户基本信息（编号、评级、合作时长）
- 展示信用额度使用情况（总额度、已用、可用）
- 分析交易记录和履约率
- 给出合作建议（如是否可提升额度）

### 场景3：财务人员分析成本
**提问**：物流板块哪些项目成本超预算了？

**AI回答**：
- 列出超预算项目清单
- 展示预算与实际成本对比表格
- 分析超预算原因（如运输成本上涨）
- 提供优化建议（如重新谈判运费）

### 场景4：法务审核合同
**提问**：最近签订的合同有哪些风险点？

**AI回答**：
- 识别高风险和中风险条款
- 详细说明风险类型（如违约责任不明、付款周期超标）
- 给出具体的合同编号和处理建议
- 推荐风险防范措施

### 场景5：自动生成报告
**提问**：帮我生成一份本月经营分析报告

**AI回答**：
- 自动生成完整的经营分析报告
- 包含整体情况、业务板块分析、重点关注事项
- 提供下月工作建议
- 支持继续补充和修改
