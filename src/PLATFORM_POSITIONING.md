# 🎯 全行BI问数平台定位说明

**定位更新时间：** 2026年2月5日  
**核心定位：** 按权限管理的全行BI问数平台

---

## 📋 平台核心定位

### 🎯 一句话定位
**全行BI问数平台** - 支持按角色知识库及权限的数据分析，支持用户定制专属数据看板

### 核心特性

1. **智能问数**
   - 自然语言提问，AI秒级响应
   - 支持复杂业务逻辑查询
   - 多轮对话深度分析

2. **权限管理**
   - 按角色分配数据访问权限
   - 行长、部门负责人、客户经理等不同角色
   - 知识库按角色隔离

3. **个性化看板**
   - 用户自定义专属数据面板
   - 拖拽排版，自由布局
   - 实时数据刷新

4. **全行覆盖**
   - 存贷汇业务全面支持
   - 6大业务模块（经营管理、存款、贷款、中间业务、客户画像、知识库）
   - 全数据源接入

## 👥 用户角色体系

### 角色分类

| 角色 | 权限级别 | 可访问模块 | 数据范围 |
|------|---------|-----------|---------|
| **行长/副行长** | 最高 | 全部模块 | 全行数据 |
| **部门负责人** | 高 | 对应业务模块 + 经营管理 | 本部门数据 |
| **支行行长** | 中高 | 全部模块 | 本支行数据 |
| **客户经理** | 中 | 存款、贷款、客户画像 | 本人客户数据 |
| **风控人员** | 中 | 贷款、经营管理 | 风险相关数据 |
| **理财经理** | 中 | 中间业务、客户画像 | 理财客户数据 |
| **柜员** | 低 | 知识库 | 无数据查询权限 |

## 📌 "我的面板"功能

### 功能定位
**个性化数据看板** - 用户专属的数据中心，支持自定义布局和实时刷新

### 核心特性

1. **一键固定**
   - 悬停在AI回答上
   - 点击"固定"按钮
   - 自动保存到"我的面板"

2. **拖拽排版**
   - 自由拖动卡片位置
   - 调整卡片大小
   - 打造个性化布局

3. **实时刷新**
   - 数据自动更新
   - 保持最新状态
   - 一键刷新全部

4. **持久化存储**
   - LocalStorage保存
   - 刷新不丢失
   - 跨会话保留

### 使用场景

**行长场景：**
- 固定"本月营收情况"
- 固定"不良贷款率趋势"
- 固定"各分行业绩排名"
- 每天打开平台即可查看核心指标

**对公客户经理场景：**
- 固定"本月新增对公客户"
- 固定"重点客户授信情况"
- 固定"对公存款排名"
- 快速掌握本人业务情况

**风控人员场景：**
- 固定"高风险客户列表"
- 固定"不良贷款分布"
- 固定"逾期预警"
- 实时监控风险指标

## 🔐 权限管理体系

### 数据权限

| 权限类别 | 说明 | 示例 |
|---------|------|------|
| **全行权限** | 可查看全行数据 | 行长、副行长 |
| **部门权限** | 可查看本部门数据 | 对公部负责人、零售部负责人 |
| **支行权限** | 可查看本支行数据 | 支行行长 |
| **个人权限** | 只能查看本人客户数据 | 客户经理 |
| **只读权限** | 只能查看知识库 | 柜员 |

### 知识库权限

**按角色分类知识库：**

```
📚 行长知识库
  - 战略规划文档
  - 监管政策法规
  - 董事会决议
  - 经营分析报告

📚 部门负责人知识库
  - 部门管理制度
  - 业务操作规范
  - 培训材料
  - 行业研究报告

📚 客户经理知识库
  - 营销话术库
  - 产品手册
  - 客户案例
  - 常见问题解答

📚 柜员知识库
  - 操作流程
  - 系统手册
  - 应急预案
  - 服务规范
```

## 💡 BI问数示例

### 行长提问示例
```
✅ "全行本月营收情况怎么样？"
→ 返回全行营收、利润、资产等经营指标

✅ "各分行业绩排名如何？"
→ 返回分行业绩对比表、排名情况

✅ "本月不良贷款率是否达标？"
→ 返回全行不良率、分支行不良率、行业分布
```

### 对公客户经理提问示例
```
✅ "我本月新增了多少对公客户？"
→ 返回本人新增客户列表、存贷款情况

✅ "某某公司的授信额度还剩多少？"
→ 返回该客户授信详情（仅限本人客户）

✅ "我的客户中哪些存款下降了？"
→ 返回本人客户存款变动分析
```

### 风控人员提问示例
```
✅ "本月有哪些高风险客户？"
→ 返回高风险客户列表、风险原因

✅ "制造业的不良贷款率是多少？"
→ 返回制造业不良率、行业对比

✅ "逾期超过90天的贷款有多少？"
→ 返回逾期贷款清单、处置建议
```

## 📊 业务价值

### 对银行的价值

1. **提升决策效率**
   - 高层：实时掌握全行经营状况
   - 中层：快速了解部门业务数据
   - 基层：便捷查询客户信息

2. **降低学习成本**
   - 自然语言提问，无需学习复杂BI工具
   - AI自动生成图表和洞察
   - 个性化看板，常用数据一目了然

3. **强化权限管理**
   - 数据按角色隔离，防止越权访问
   - 知识库分级管理，保护敏感信息
   - 操作日志记录，可追溯审计

4. **促进数据驱动**
   - 随时随地查询数据
   - AI主动发现业务趋势
   - 数据固定到看板，持续关注
