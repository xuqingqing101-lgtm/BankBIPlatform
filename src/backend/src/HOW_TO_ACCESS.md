## 🔌 连接后端

### 1. 确保后端已启动

```bash
cd backend
RUN.bat
```

**成功标志：**
```
========================================
🏦 银行智能AI分析平台已启动
========================================
API地址: http://localhost:8080/api
```

### 2. 测试后端连接

```bash
curl http://localhost:8080/api/health
```

**应该返回：**
```json
{
  "status": "UP",
  "message": "银行智能AI分析平台运行正常"
}
```

### 3. 前端自动连接

前端会自动连接到：`http://localhost:8080/api`

如果后端不可用，前端会显示模拟数据并提示连接后端。

## 🆘 常见问题

### Q1: 找不到前端页面？

**A:** 前端在Figma Make中自动运行：
1. 查看Figma Make的预览窗口
2. 或访问 `http://localhost:5173`
3. 确保Figma Make项目已打开

### Q2: 页面显示但功能不工作？

**A:** 检查后端是否运行：
```bash
cd backend
RUN.bat
```

### Q3: API连接失败？

**A:** 前端会自动降级到模拟数据，你会看到提示：
```
💡 提示：后端服务暂时不可用，显示的是模拟数据。
```

**解决方法：**
1. 启动后端：`cd backend && RUN.bat`
2. 测试连接：`curl http://localhost:8080/api/health`
3. 刷新前端页面

### Q4: 如何修改前端代码？

**A:** 前端代码在项目根目录：
- `/App.tsx` - 主入口
- `/components/` - 所有组件
- `/lib/api.ts` - API配置
- `/styles/globals.css` - 全局样式

Figma Make会自动热重载！

### Q5: 如何查看前端日志？

**A:** 按F12打开浏览器开发者工具：
- **Console** - 查看日志和错误
- **Network** - 查看API请求
- **Application** - 查看localStorage
